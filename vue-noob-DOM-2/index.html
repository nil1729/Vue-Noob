Using VueJS to Interact with the DOM
1. Understanding VueJS Templates

{{ title }}

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-1"> <p>{{ title }}</p> </div> <script> new Vue({ el: '#dom-1', data: { title: 'Hello Vue JS' } }) </script>
2. How the VueJS Template Syntax and Instance Work Together

{{ sayHello() }}

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-2"> <p>{{ sayHello() }}</p> </div> <script> new Vue({ el: '#dom-2', data: { title: 'Hello Vue JS' }, methods: { sayHello() { return this.title } } }) </script>
