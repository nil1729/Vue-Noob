2. How the VueJS Template Syntax and Instance Work Together

{{ sayHello() }}

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-2"> <p>{{ sayHello() }}</p> </div> <script> new Vue({ el: '#dom-2', methods: { sayHello() { return 'Hello Vue JS' } } }) </script>
3. Accessing Data in the Vue Instance

{{ sayHello() }}

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-3"> <p>{{ sayHello() }}</p> </div> <script> new Vue({ el: '#dom-3', data: { title: 'Hello Vue JS' }, methods: { sayHello() { return this.title } } }) </script>
4. Binding to Attributes

{{ sayHello() }} - Google

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-4"> <p class="lead">{{ sayHello() }} - <a v-bind:href="link" target="_blank">Google</a></p> </div> <script> new Vue({ el: '#dom-4', data: { title: 'Hello Vue JS', link: 'https://google.com' }, methods: { sayHello() { return this.title } } }) </script>
