Using VueJS to Interact with the DOM - 4
1. Dynamic Styling with CSS Classes - Basics
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-1"> <div class="p-4 border border-secondary" @click="bgRed = !bgRed" :class="{'bg-primary': bgRed}"> </div> </div> <script> new Vue({ el: '#dom-1', data: { bgRed: false } }); </script>
2. Dynamic Styling with CSS Classes - Using Objects
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-2"> <div class="p-4 border border-secondary" @click="bgRed = !bgRed" :class="divClasses"></div> </div> <script> new Vue({ el: '#dom-2', data: { bgRed: false }, computed: { divClasses: function () { return { 'bg-primary': this.bgRed, 'bg-info': !this.bgRed } } } }); </script>
3. Dynamic Styling with CSS Classes - Using Names



<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-3"> <div class="p-4 border border-secondary" @click="bgRed = !bgRed" :class="{'bg-primary': bgRed}"> </div> <br> <div class="p-4 border border-secondary" :class="bgInput"></div> <br> <div class="p-4 border border-secondary" :class="[bgInput, {'bg-primary': bgRed}]"></div> <br> <div class="form-group"> <label class="form-control-label" for="inputSuccess1">Type Bootsrap Classes</label> <input v-model="bgInput" type="text" class="form-control"> </div> </div> <script> new Vue({ el: '#dom-3', data: { bgRed: false, bgInput: 'bg-success' } }); </script>
4. Setting Styles Dynamically (without CSS Classes)

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-4"> <div class="border p-2 border-warning"> <div class="p-4" :style="{backgroundColor: color1}"></div> <div class="form-group mt-2"> <label class="form-control-label" for="inputSuccess1">Enter Color Code or Name</label> <input v-model="color1" type="text" class="form-control"> </div> </div> <br> <div class="border p-2 border-warning"> <div class="p-4" :style="myStyles"></div> <div class="form-group mt-2"> <label class="form-control-label" for="inputSuccess1">Enter Color Code or Name</label> <input v-model="color2" type="text" class="form-control"> </div> <div class="form-group mt-2"> <label class="form-control-label" for="inputSuccess1">Enter Width (px)</label> <input v-model="width" type="text" class="form-control"> </div> </div> </div> <script> new Vue({ el: '#dom-4', data: { color1: 'pink', width: 100, color2: 'lightblue' }, computed: { myStyles: function () { return { backgroundColor: this.color2, width: `${this.width}px` } } } }) </script>
5. Styling Elements with the Array Syntax
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> <div id="dom-5"> <div class="p-4" :style="[myStyles, {height: width + 'px'}]"></div> <div class="form-group mt-2"> <label class="form-control-label" for="inputSuccess1">Enter Color Code or Name</label> <input v-model="color" type="text" class="form-control"> </div> <div class="form-group mt-2"> <label class="form-control-label" for="inputSuccess1">Enter Width (px)</label> <input v-model="width" type="text" class="form-control"> </div> </div> <script> new Vue({ el: '#dom-5', data: { color: 'khaki', width: 150, }, computed: { myStyles: function () { return { backgroundColor: this.color, width: `${this.width}px` } } } }) </script>
